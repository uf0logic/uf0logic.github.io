# Assets Folder

This folder contains image assets for the website.

## Required Images

The following images need to be downloaded from Figma and placed in this folder:

1. **hero-bg.png** - Hero section background image
   - Original URL: `http://localhost:3845/assets/c97c631293ca34d347c0ac3d78e49965a2096878.png`
   - Used in: Hero section and Services section

2. **case-1.png** - Case study 1 image (Море перемен)
   - Original URL: `http://localhost:3845/assets/d256b0b7f01480379b52ec2e0f0fad10cfb20943.png`
   - Used in: Cases section, first case card

3. **case-2.png** - Case study 2 image (Karaoke star hits)
   - Original URL: `http://localhost:3845/assets/9347313de14b027df327db1b4d85ac9407a91827.png`
   - Used in: Cases section, second case card

4. **case-3.png** - Case study 3 image (CD Projekt Red)
   - Original URL: `http://localhost:3845/assets/21508c03701ed3b7193fefb101f2d2bf1fde0b86.png`
   - Used in: Cases section, third case card

## How to Download

1. Ensure the Figma MCP server is running on `http://localhost:3845`
2. Open the Figma design file
3. Export the images from Figma or use the MCP server URLs to download them
4. Save them with the filenames listed above in this `assets` folder

Alternatively, you can manually export the images from Figma and save them with the correct names.
